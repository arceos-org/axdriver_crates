# axdriver_net

[![Crates.io](https://img.shields.io/crates/v/axdriver_net)](https://crates.io/crates/axdriver_net)
[![Docs.rs](https://docs.rs/axdriver_net/badge.svg)](https://docs.rs/axdriver_net)
[![CI](https://github.com/arceos-org/axdriver_crates/actions/workflows/deploy.yml/badge.svg?branch=main)](https://github.com/arceos-org/axdriver_crates/actions/workflows/deploy.yml)

Common traits and types for network device (NIC) drivers in `no_std` environments.

Part of the [axdriver_crates](https://github.com/arceos-org/axdriver_crates) workspace.

## Features

- `ixgbe` – Intel 10GbE NIC driver.
- `fxmac` – FXMAC ethernet driver for Phytium Pi.
